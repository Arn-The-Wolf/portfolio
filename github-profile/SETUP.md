# GitHub Profile README — Setup

Your iconic profile README lives in this folder. Deploy it to your **special GitHub profile repo**.

## Steps

1. Open or create the repo: [github.com/Arn-The-Wolf/Arn-The-Wolf](https://github.com/Arn-The-Wolf/Arn-The-Wolf)

2. Copy these files into that repo:
   - `README.md` → root `README.md`
   - `.github/workflows/snake.yml` → same path (enables the contribution snake animation)

3. Commit & push to `main`

4. Visit [github.com/Arn-The-Wolf](https://github.com/Arn-The-Wolf) — your profile README appears at the top.

## After first push

- Go to **Actions** tab in `Arn-The-Wolf/Arn-The-Wolf`
- Run **Generate Snake** workflow manually once
- The snake SVG will appear in the `output` branch and show on your README
